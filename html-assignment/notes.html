HTML tags & descriptions

  1. <table>:

    This tag is used to create tables in HTML. Tables are used to organize and display data in rows and columns.

  2. <p>:

    Represents a paragraph of text in an HTML document. It's used to structure and format text content.

  3. <img>:

    Inserts an image into an HTML document. It requires the "src" attribute to specify the image file's URL.

  4. <a>:

    Defines a hyperlink, linking to another web page or resource. It requires the "href" attribute to specify the URL of the link destination.

  5. <ul>:

    Represents an unordered list in HTML. It's used to create a list of items with bullet points.
