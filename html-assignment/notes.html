HTML Notes

HTML

HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser.

Headings

HTML has six levels of headings, which are used to structure the content on a webpage.

Paragraphs

The <p> tag is used to define paragraphs in HTML.

Lists

HTML supports ordered (<ol>) and unordered (<ul>) lists.

Images

Images can be embedded in HTML using the <img> tag.
